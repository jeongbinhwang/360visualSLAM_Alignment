ALL codes are based on Stella vSLAM.

To use door_corner_based_alignment, first, you have to run Stella vSLAM.
https://stella-cv.readthedocs.io/en/latest/

To use marker_based_alignment, we edited Stella vSLAM to refine 360 visual SLAM output using ArUco markers.
https://github.com/sbagh_ncstate/stella_vslam_marker_priors
This is Shitikantha's 360 visual SLAM code.
